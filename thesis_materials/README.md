# Thesis Materials

本目录用于保存可直接迁移到毕业论文中的 LaTeX 素材，包含：

- `main.tex`：论文主体草稿
- `references.bib`：参考文献 BibTeX
- `tables/*.tex`：实验结果表格，可直接 `\input{}`

建议用法：

1. 如果学校提供了毕业论文模板，将 `main.tex` 中正文内容迁移到模板对应章节。
2. 保留 `tables/` 目录中的结果表，避免手工重复录入指标。
3. 将 `EXPERIMENTS.md` 作为实验记录底稿，将本目录作为论文排版底稿。

当前文稿重点覆盖：

- 研究背景与意义
- 多源数据融合方法设计
- 不平衡数据优化实验
- 威胁情报模拟与决策层融合实验
- 消融实验与复现说明
- 结果分析、局限性与未来工作
